е®һйӘҢеҶ…е®№

  • е®һзҺ°и®Ўз®—еҷЁзұ»дёӯlpгҖҒrpе’ҢoperateеҮҪж•°
  • е®Ңе–„и®Ўз®—еҷЁзұ»evaluateеҮҪж•°пјҢеўһеҠ еҜ№иҫ“е…Ҙзҡ„еҗҲжі•жҖ§жЈҖжҹҘпјҢеҢ…жӢ¬ж»ӨжҺүжүҖжңүйқһжі•иҫ“е…ҘеҸҠеӨ„зҗҶе·ҰеҸіжӢ¬еҸ·дёҚй…ҚеҜ№зҡ„иҫ“е…Ҙ
  • зј–еҲ¶еә”з”ЁзЁӢеәҸжөӢиҜ•иҝҷдёӘи®Ўз®—еҷЁ

зЁӢеәҸиҰҒжұӮ

вҖӢ еҲ©з”Ёж ҲеҺҹзҗҶпјҢи®ҫи®ЎдёҖдёӘеҸҜд»Ҙи®Ўз®—жүҖиҫ“е…ҘиЎЁиҫҫејҸзҡ„зЁӢеәҸпјҢиҰҒжұӮеҸҜд»Ҙи®Ўз®—жӯЈж•°пјҢиҙҹж•°пјҢе°Ҹж•°зҡ„еҠ еҮҸд№ҳйҷӨиҝҗз®—е’ҢжӢ¬еҸ·иҝҗз®—пјҢд»Ҙ#з»“жқҹиҫ“е…ҘгҖӮ
